## v1.0.1 (2026-06-23)

🔒 **安全加固 + 国际化完善**

### 安全修复

- 系统热重载端点 `/api/system/reload` 新增登录认证保护，防止未授权调用。
- 语言包和主题包上传接口增加文件名正则校验（仅允许 `[a-zA-Z0-9_-]+`），防止路径穿越攻击。
- 远程同步触发接口 (`/api/sync/remote/now`) 现在正确传递 `user_id` 到底层引擎，防止跨用户数据同步。
- 修改联系方式 (`change-contact`) 两个端点中将 f-string 拼接 SQL 字段名的写法替换为 `if/else` 安全分支，消除潜在的 SQL 注入风险。

### 国际化

- 简体中文 (`zh-CN.json`)、英文 (`en-US.json`)、繁体中文 (`zh-TW.json`) 统一补齐至 211 个翻译 key。
- 新增 sync.* 同步任务模块翻译（状态、触发方式、时间等 14 个 key）。
- 新增 compose.* 写邮件模块翻译（保存草稿、取消确认、发送中等 5 个 key）。
- 新增 common.close 和 tasks.confirmDelete 翻译。

### 修复

- 修正 `routes_system.py` 健康检查端点中版本号，与主应用版本保持一致。

## v1.0.0 (2026-06-23)

🎉 **首个完整版本发布！**

### 新增

- **多邮箱统一管理** —— 支持同时添加和管理多个邮箱账户，聚合收件箱与未读汇总。
- **12 家服务商自动配置** —— Google、Microsoft、腾讯、阿里、Apple、网易、Yahoo、TOM、新浪、搜狐、Zoho、139/联通沃邮箱自动匹配 IMAP/SMTP。
- **聚合收件箱** —— 所有账户邮件汇集同一视图，支持按账户筛选。
- **未读汇总** —— 快速浏览所有账户未读邮件数量与列表。
- **邮件搜索与标签** —— 全文搜索邮件，支持自定义标签分类。
- **远程内容安全控制** —— 默认不加载远程图片和脚本，用户确认后在沙箱 iframe 中显示 HTML 邮件。
- **SMTP TLS/SSL 发信** —— 所有外发邮件默认加密通道传输。
- **PGP 端到端加密** —— 内置 PGP 密钥生成、导入/导出、加密发送和解密阅读。
- **日历** —— 月/周/日视图，支持日程创建、编辑、提醒。
- **通讯录** —— 联系人管理，支持分组、搜索和空状态提示。
- **任务** —— 待办事项管理，标记完成状态和截止日期。
- **便签** —— 轻量笔记，随手记录与编辑。
- **CalDAV / CardDAV 同步** —— 日历和通讯录可通过标准协议与第三方客户端同步。
- **Google / Microsoft Graph 同步** —— 双向同步 Google 日历、Outlook 日历和通讯录。
- **OAuth2 一键登录** —— 支持 Google、Microsoft、QQ、Yahoo、Zoho OAuth2 授权。
- **Thunderbird 全量数据迁移** —— 一键导入 profile、prefs.js 和本地邮件目录。
- **多设备远程自动同步** —— 同步接口支持多设备间数据一致性。
- **响应式布局** —— 适配 PC、平板和手机三端。
- **日间/夜间主题** —— 内置双主题，支持一键切换。
- **语言包系统** —— 默认简体中文，内置繁体中文和英文，支持社区扩展。
- **插件社区** —— 本地和在线插件生态，安装前自动校验插件清单。
- **翻译服务** —— 集成 MyMemory、LibreTranslate、Lingva、自定义 API 和 OpenAI 兼容大模型接口。
- **WebDAV 备份** —— 一键备份本地数据到 WebDAV 服务器。
- **热更新** —— 升级版本无需重建 Docker 镜像。
- **验证码邮件/短信真实发送** —— 注册和敏感操作通过邮件/短信验证。
- **PBKDF2-SHA256** —— 总账户密码使用强散列算法存储。
- **Fernet 加密** —— 所有邮箱账户密钥本地加密保存。
- **Docker 一键部署** —— `docker compose up -d` 即可启动。
- **多架构支持** —— 支持 linux/amd64 和 linux/arm64 镜像构建。
